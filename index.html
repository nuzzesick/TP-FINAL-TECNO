Conocé Processing

Un lenguaje de programación y entorno de desarrollo integrado de código abierto para la enseñanza y producción de proyectos multimedia e interactivos de diseño digital.

Creadores

Ben Fry launch

Benjamin Fry (nacido en 1975) es un experto estadounidense en visualización de datos. Es director de Fathom, una consultora de diseño y software en Boston, Massachusetts. También es co-desarrollador de Processing, un lenguaje de programación de código abierto y un entorno de desarrollo integrado (IDE) creado para las comunidades de artes electrónicas y diseño visual con el propósito de enseñar los conceptos básicos de la programación de computadoras en un contexto visual. El entorno de diseño de Processing ganó una Golden Nica de Prix Ars Electronica en 2005.

Comenzar a programar en Processing

Veamos cómo dibujar nuestro primer rectángulo en pantalla
            
void setup() {
  size(400,400);
}

void draw() {
  rect(width/2,height/2,100,50);
}
Listo? como podemos ver es bastante fácil
Ahora que aprendiste a crear un rectángulo, te enseñaremos la sintaxis para crear otras figuras.
  • codeellipse()
    ellipse(56, 46, 55, 55);



    Dibuja una elipse (ovalada) en la pantalla. Una elipse con igual ancho y alto es un círculo. De manera predeterminada, los dos primeros parámetros establecen la ubicación, y los parámetros tercero y cuarto establecen el ancho y la altura de la forma. El origen se puede cambiar con la función ellipseMode ().
  • codeline()
    line(30, 20, 85, 75);



    Dibuja una línea (una ruta directa entre dos puntos) a la pantalla. La versión de line () con cuatro parámetros dibuja la línea en 2D. Para colorear una línea, use la función trazo (). No se puede llenar una línea, por lo tanto, la función fill () no afectará el color de una línea. Las líneas 2D se dibujan con un ancho de un píxel de forma predeterminada, pero esto se puede cambiar con la función strokeWeight (). La versión con seis parámetros permite colocar la línea en cualquier lugar dentro del espacio XYZ. Dibujar esta forma en 3D con el parámetro z requiere el parámetro P3D en combinación con el tamaño () como se muestra en el ejemplo anterior.
  • codepoint()
    point(30, 20);



    Dibuja un punto, una coordenada en el espacio en la dimensión de un píxel. El primer parámetro es el valor horizontal para el punto, el segundo valor es el valor vertical para el punto y el tercer valor opcional es el valor de profundidad. Dibujar esta forma en 3D con el parámetro z requiere el parámetro P3D en combinación con el tamaño () como se muestra en el ejemplo anterior.
  • codetriangle()
    triangle(30, 75, 58, 20, 86, 75);



    Un triángulo es un plano creado conectando tres puntos. Los dos primeros argumentos especifican el primer punto, los dos argumentos centrales especifican el segundo punto y los dos últimos argumentos especifican el tercer punto.
